v0 Portfolio Website

This project is a personal portfolio website created using v0 (AI-powered website generation) and deployed on Vercel. It was built as a learning project to explore new web technologies, modern UI patterns, and AI-assisted development workflows.

The website showcases projects, skills, and professional experience while serving as a hands-on experiment in using AI tools to accelerate front-end development and improve productivity.

Features:

- Built using v0 (AI-assisted development)
- Deployed with Vercel for fast performance
- Responsive and modern UI
- Project and skill showcase
- Focused on learning and experimenting with new technologies
